Uninstallation

Last modified: July 1, 2000
Created: May 7, 2000

0. このドキュメントの目的

このドキュメントは，KINOBOARDS version 1.0 release 7をアンインストールするための手順解説書です．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.2） です．

1. データの保存

アンインストールすると，作成した掲示板や，各掲示板に登録されているメッセージも含めて全て回復不可能になります． そのため，できればアンインストールの前に，非公開ディレクトリ: kbdataディレクトリのバックアップを取っておいてください． これがあれば，将来掲示板として復活させることも可能です．

2. アンインストール手順

KINOBOARDS version 1.0 release 7をアンインストールするためには，プロバイダ（ウェブサーバ）に転送した以下のディレクトリを削除してください．

  • 公開ディレクトリ: kbディレクトリ
  • CGIプログラム: index.cgiファイル
  • 非公開ディレクトリ: kbdataディレクトリ

ただしあなたの環境に依っては，CGIプログラム: index.cgiファイルは，公開ディレクトリ: kbディレクトリに含まれているかもしれません． この場合，kbディレクトリを削除すれば，index.cgiファイルも削除されます．

非公開ディレクトリ: kbdataディレクトリを削除すると，掲示板や，各掲示板に登録されているメッセージも含めて全て削除されます． 御注意ください．
