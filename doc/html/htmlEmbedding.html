KINOBOARDS HTML Embedding

Last modified: August 10, 2000
Created: August 7, 2000

0. このドキュメントの目的

きのぼずが表示する画面の見た目は，UIディレクトリの中に収められているXMLファイル（以後UI定義ファイルと呼ぶことにします）を変更することにより，カスタマイズすることができます． UI定義ファイルは基本的にXHTMLそのままですが，一部きのぼず固有のタグを入れることができます． きのぼずはこのタグを見て，ジャンプメニューやメッセージのスレッドなどを合成し，最終的な画面を生成しています．

このドキュメントは，UI定義ファイルの中で使える，きのぼず固有のタグについて説明するためのものです． きのぼず固有のタグはUI定義ファイルに埋め込むものですので，適宜 KINOBOARDS UI Definition ドキュメントも参照してください．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.3以降） です．

きのぼず固有のタグ（以下，きのぼずタグと呼ぶことにします）は，大まかに，以下の4種類に分けることができます． それぞれ説明します．

1. 共通タグ
どのUI定義ファイルの中にでも置くことができます． タグ名はc_で始まります．
2. 掲示板タグ
表示している掲示板に関する情報を表示するためのタグです． 掲示板が1つに定まらないUI定義ファイル（例えば掲示板一覧を表示するためのBoardList.xmlなど）の中には置くことができません． タグ名はb_で始まります．
3. 固有タグ
固有のUI定義ファイルでしか使えないタグです． この種類のタグは，指定されたUI定義ファイルの中にしか置けません （そのファイルの中であれば，任意の位置に置くことができます）．
4. システムタグ
きのぼずシステムが動くために必須なタグです． この種類のタグの移動や追加，削除は慎重にお願いします． タグ名はs_で始まります．

1. 共通タグ

どのUI定義ファイルの中にでも置くことができます． タグ名はc_で始まります．

<kb:c_anchor />

<kb:c_anchor comment="リンク文字列" url="リンク先URL" />

このタグが置かれた位置に，リンク先URLへのリンクを挿入します． aエレメントです． UI定義ファイル中にそのまま<a href="...と書いても構いませんが，このきのぼずタグを利用すると，必要に応じてaccesskeyが補われます．

<kb:c_auth_user />

<kb:c_auth_user />

このタグが置かれた位置に，アクセス中のユーザ名を挿入します． 単なる文字列です．

<kb:c_board_lastmod />

<kb:c_board_lastmod board="掲示板名" />

このタグが置かれた位置に，指定された掲示板の最終更新時刻を挿入します． 単なる文字列です．

<kb:c_board_name />

<kb:c_board_name board="掲示板名" />

ex. <kb:c_board_name board="test" />

このタグが置かれた位置に，指定された掲示板の名前を挿入します． 単なる文字列です． <kb:b_board_name />との違いは，掲示板名を明示する点です．

<kb:c_board_link />

<kb:c_board_link com="コマンド" board="掲示板名" />

ex. <kb:c_board_link com="message-thread" board="test" />

このタグが置かれた位置に，掲示板のタイトル一覧，メッセージ一覧へのリンクをを挿入します． aエレメントの他に，最終更新時刻なども表示されます． コマンドとしては，以下のものを指定してください．

title-thread
タイトル一覧（スレッド）へのリンク
title-bydate
タイトル一覧（日付順）へのリンク
message-thread
メッセージ一覧（スレッド）へのリンク
message-bydate
メッセージ一覧（日付順）へのリンク

<kb:c_board_link_all />

<kb:c_board_link_all />

このタグが置かれた位置に，各掲示板のタイトル一覧へのリンクをを挿入します． ulエレメントです． 最終更新時刻なども表示されます．

<kb:c_command />

<kb:c_command comment="リンク文字列" tag1="value1" tag2="value2" ... />

ex. <kb:c_command comment="検索" c="s" b="test" />

このタグが置かれた位置に，きのぼずのコマンドへのリンクを挿入します． aエレメントです． tagとvalue部分には，きのぼずに渡すクエリを書いてください． 上記の例であれば，kb.cgi?c=s&b=testというリンクを生成します．

<kb:c_exec_date_time />

<kb:c_exec_date_time />

このタグが置かれた位置に，現在時刻を挿入します． 単なる文字列です．

<kb:c_func_link />

<kb:c_func_link />

このタグが置かれた位置に，認証関連の各機能を呼び出すためのリンクを挿入します． dlエレメントです． ログイン画面，ユーザ情報画面，新規登録画面へのリンクが（もし権限があれば）表示されます．

<kb:c_help />

<kb:c_help type="タイプ" />

ex. <kb:c_help type="boards" />

このタグが置かれた位置に，ヘルプ画面へのリンクを挿入します． aエレメントです． タイプとしては，以下のものを指定してください．

boards
掲示板一覧画面のヘルプ
titles
タイトル一覧画面のヘルプ
thread
スレッドについてのヘルプ
message
メッセージ表示画面のヘルプ
icon
コマンドアイコンのヘルプ
post
メッセージ登録画面のヘルプ

<kb:c_icon />

<kb:c_icon icon="アイコンファイル名" alt="説明文字列" />

ex. <kb:c_icon icon="org_help.gif" alt="ヘルプ" />

このタグが置かれた位置に，アイコンを挿入します． imgエレメントです． メッセージアイコンに限らず，iconディレクトリ内に置かれた画像ファイルであれば指定できます． アイコン名ではなく，アイコンファイル名を指定してください．

<kb:c_icon_msg />

<kb:c_icon_msg icon="アイコン名" />

ex. <kb:c_icon_msg icon="new!" />

このタグが置かれた位置に，メッセージアイコンを挿入します． imgエレメントです． アイコン名を指定します．

<kb:c_img />

<kb:c_img src="ファイル名" alt="説明文字列" width="幅" height="高さ" />

ex. <kb:c_img src="boards.png" alt="タイトル一覧ページ" width="474" height="256" />

このタグが置かれた位置に，きのぼずのimgディレクトリ内に置かれた画像ファイルを挿入します． imgエレメントです． ファイルはimgディレクトリに置かれたものでなければなりません． 具体的には，kb.cgiの先頭で指定した$KB_RESOURCE_URL下のimgディレクトリ直下です． 説明文字列，高さ，幅は必ず指定してください．

<kb:c_page_link />

<kb:c_page_link />

このタグが置かれた位置に，一覧表示時のページ移動用情報（最新，次，逆順，など）を挿入します． kbPageLinkというクラスのpエレメントです． 「共通タグ」に分類されていますが，一覧表示用のUI定義ファイルに埋め込まなければ，意味がありません （その他のUI定義ファイルに埋め込んだ場合は，何も表示されません．）

<kb:c_remote_info />

<kb:c_remote_info />

このタグが置かれた位置に，アクセスユーザのリモート情報を挿入します． 単なる文字列です． リモート情報は，ユーザのアクセス元ホスト情報（ホスト名もしくはIPアドレス）です． BasicAuth認証を利用している場合は，認証ユーザ名も入ります．

<kb:c_site_name />

<kb:c_site_name />

このタグが置かれた位置に，この掲示板システムの名前を挿入します． 単なる文字列です． 具体的には，kb.phで宣言された，$SYSTEM_NAMEが入ります．

<kb:c_status />

<kb:c_status />

このタグが置かれた位置に，きのぼずのステータスを挿入します． kbStatusというクラスのpエレメントです． ステータスは，認証済みであればユーザ名と，アクセス中の掲示板，現在時刻から成ります．

<kb:c_top_menu />

<kb:c_top_menu />

このタグが置かれた位置に，きのぼずのトップメニューを挿入します． kbTopMenuというクラスのdivエレメントです． トップメニューはTOP，HELP，OPENなどのリンクと， タイトル一覧やメッセージ一覧を表示するためのフォームから成ります． 参照権限がないユーザに対しては，フォームが表示されません．

2. 掲示板タグ

表示している掲示板に関する情報を表示するためのタグです． 掲示板が1つに定まらないUI定義ファイル（例えば掲示板一覧を表示するためのBoardList.xmlなど）の中には置くことができません． タグ名はb_で始まります．

（実際には，置いてもエラーにはなりません． ただ何も表示されないだけです．）

このタイプのきのぼずタグは，属性を取りません．

<kb:b_all_icon />

<kb:b_all_icon />

このタグが置かれた位置に，該当掲示板で利用可能な全てのメッセージアイコンが挿入されます． ulエレメントです． 各メッセージアイコンはliエレメントの中に入っています．

<kb:b_board_header />

<kb:b_board_header />

このタグが置かれた位置に，該当掲示板のヘッダを挿入します． 具体的には，各掲示板ディレクトリ内のkb.boardファイルが展開されます． kb.boardファイルの中にも，きのぼずタグを挿入することができます

<kb:b_board_name />

<kb:b_board_name />

このタグが置かれた位置に，該当掲示板の掲示板名を挿入します． 単なる文字列です．

<kb:b_post_entry_form />

<kb:b_post_entry_form />

このタグが置かれた位置に，該当掲示板にオリジナルメッセージを書き込むためのフォームを挿入します． formエレメントです．

<kb:b_search_article_form />

<kb:b_search_article_form />

このタグが置かれた位置に，該当掲示板のメッセージを検索するためのフォームを挿入します． formエレメントです．

4. システムタグ

きのぼずシステムが動くために必須なタグです． この種類のタグの移動や追加，削除は慎重にお願いします． タグ名はs_で始まります．

このタイプのきのぼずタグは，属性を取りません．

<kb:s_address />

<kb:s_address />

このタグが置かれた位置に，管理者名およびきのぼずの著作権表示を挿入します． このタグは移動しても構いませんし，削ってしまっても構いません．

<kb:s_title />

<kb:s_title />

このタグが置かれた位置に，XHTMLのヘッダ項目を挿入します． XHTMLのヘッダエレメントの中から動かさないでください．
