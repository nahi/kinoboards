1. jcode.plとの相性

CGIとperl4との組み合わせの場合，jcode.plとの相性が悪いのか， 文字の取りこぼしまたはサーバエラーが頻繁に起こることがあります．

だったんですが， 加瀬さん に御協力頂き，R2.3でcgi.pl.jcodeを修正したら， なんだかエラーが出なくなってしまいました． 原因をはっきりとつきとめたわけではないため， 今後また，何かの拍子に再発するおそれがあります．

その場合，まず/usr/local/bin/kcもしくは/usr/local/bin/nkf (あるいはそれに類する漢字コード変換機能を持つUNIXプログラム) が存在することを確認し，cgi.pl.outcomをcgi.plに置き換えてください． cgi.plは，配布パッケージでは，cgi.pl.jcodeと同じものになっています．

rm cgi.pl
cp -p cgi.pl.outcom cgi.pl
次にcgi.pl.outcomの中の 
$KC2IN = "/usr/local/bin/nkf -e";
$KC2OUT = "/usr/local/bin/nkf -j";
を書き換えて，EUC，およびJISに変換するコマンドを指定してください．
