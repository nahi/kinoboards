KINOBOARDS UI Definition

Last modified: August 7, 2000
Created: August 7, 2000

0. このドキュメントの目的

きのぼずが表示する画面は，UIディレクトリの中に収められているXMLファイル（以後UI定義ファイルと呼ぶことにします）を元に生成されています． このドキュメントは，各種UI定義ファイルについて説明するためのものです．

UI定義ファイルは基本的にXHTMLそのままですが，一部きのぼず固有のタグを入れることができます． きのぼず固有タグについては， KINOBOARDS HTML Embedding ドキュメントを参照してください．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.3以降） です．

きのぼずのUI定義ファイルには，以下のものがあります． 個々について説明します．

  • BoardList.xml: 掲示板一覧
  • ThreadTitle.xml: タイトル一覧（スレッド）
  • SortTitle.xml: タイトル一覧（日付順）
  • ThreadArticle.xml: メッセージ一覧（スレッド）
  • SortArticle.xml: メッセージ一覧（日付順）
  • ShowArticle.xml: 単一メッセージ表示
  • ShowThread.xml: リプライメッセージをまとめて表示
  • PostNewEntry.xml: 新規メッセージ投稿
  • PostReplyEntry.xml: リプライメッセージの投稿
  • PostPreview.xml: メッセージ投稿の確認
  • PostExec.xml: メッセージ投稿の実施結果
  • SupersedeEntry.xml: メッセージ訂正
  • SupersedePreview.xml: メッセージ訂正の確認
  • SupersedeExec.xml: メッセージ訂正の実施結果
  • DeletePreview.xml: メッセージ削除の確認
  • DeleteExec.xml: メッセージ削除の実施結果
  • SearchArticle.xml: メッセージ検索
  • ShowIcon.xml: メッセージアイコン一覧
  • Login.xml: ログイン
  • UserEntry.xml: 新規ユーザの登録
  • UserConfig.xml: ユーザ情報の変更
  • AdminConfig.xml: 管理者パスワード設定変更
  • BoardEntry.xml: 新規掲示板作成
  • BoardConfig.xml: 掲示板設定変更
  • Help.xml: ヘルプ
  • Fatal.xml: エラー
