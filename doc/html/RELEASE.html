1. 変更点

ドキュメント作成が遅れに遅れた挙げ句，やっとR7.0をリリースすることができました． お待ち頂いた皆さん（もし居れば）ごめんなさい．

R7では数々の機能を盛り込みました． 機能概要 に(*)として書きましたので，参照してみてください． 一番大きな機能追加はやはり，認証関連でしょうか． セキュリティホールを仕込んでしまっていないことを祈ります．．． しばらく使われて，その安全性が検証されるまでは， 大切な情報を守るために使うのは避けたほうがいいかもしれません． （え，最初からウェブでそんなことするヤツは居ない? ^^;）
