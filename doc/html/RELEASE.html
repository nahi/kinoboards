0. このドキュメントの目的

このドキュメントは，KINOBOARDS version 1.0 release 7のリリースノートです． 本文書が含まれているリリースパッケージについての解説です．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.3） です．

1. 変更点

R7.3での変更点は以下の通りです．

  • Customizeドキュメントを追加．
  • きのぼず固有タグ の属性の書式を変更． 具体的には，従来すべて「var="..."」の形だったものが，「comment="..." icon="..."」など，分解され，かつ適切な名前に変わっています． 従来の UI定義ファイル およびkb.boardのうち，「var="..."」の形で属性を指定しているものは，全て書き直さなければなりません． すいません． 書式の詳細については， KINOBOARDS HTML Embedding を参考にしてください．
  • ユーザにHTTP-Cookiesの利用を強要する（URL認証の利用を許さない）オプションを追加．
  • 訂正可能か否かを指定するオプションと，削除可能か否かを指定するオプションを分離．
  • きのぼず固有タグ の種類を追加． <c_command />，<c_board_name />，<c_board_lastmod />．
  • 否定検索機能を追加．
  • デフォルトのスタイルシートをsummer2k.cssに変更． 作ってくださったU-Zさんに感謝!
  • メイル中の[url:kb:...]を，本来のURLに展開するようにした．
  • URL認証方式を利用していた場合に，管理者パスワードおよびユーザ情報の設定ができないというバグを修正．
  • サーバMac + MacPerlで動かなかったバグを修正（しきれてないかも）． penguin-19さんに感謝!

R7.2からの変更点の詳細については， Changes: 変更履歴 を参照してください．
