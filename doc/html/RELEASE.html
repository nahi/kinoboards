0. このドキュメントの目的

このドキュメントは，KINOBOARDS version 1.0 release 7のリリースノートです． 本文書が含まれているリリースパッケージについての解説です．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.1） です．

1. 変更点

R7.1での変更点は以下の通りです．

  • セキュリティ保護のため，パスワード変更時に，旧パスワードを入力しなければならないようにした． R7.0では旧パスワードなしに変更できましたが，これは大きなセキュリティホールです （詳しくはコラム: R7における認証方式を御覧ください．
  • Sageさんの指摘で，新規掲示板作成画面，掲示板設定変更画面において，kb.confを利用するか否かが指定できないというバグを修正した． 従来は「kb.confは使わない」ようにしか指定できませんでした．
  • 日付検索において，FROMを指定するとTOが無視される（指定されていてもされていなくても）というバグを修正した．
  • Sageさんの指摘で，ユーザ情報変更時に，メイルアドレス重複をチェックしていないというバグを修正した．
  • キーボードショートカットを表示しないようにするカスタマイズオプションを用意した．
  • ひろゆきさんの指摘で，スレッドの並べ替え方式を1つ追加した． 従来はオリジナルメッセージの投稿時間で並べ替えていたが，最後にリプライのついた時間で並べ替えることもできるようになった． カスタマイズオプションで指定する．
  • 車に轢かれた猫の指摘で，投稿時間検索用のコマンドとして「tpi」を追加した． 「..../index.cgi?b=掲示板&c=s&tpi=yyyy/mm/dd」で，yyyy/mm/ddに投稿されたメッセージのみを検索・表示する．

R7.0からの変更点の詳細については， Changes: 変更履歴 を参照してください．
