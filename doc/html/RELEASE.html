0. このドキュメントの目的

このドキュメントは，KINOBOARDS version 1.0 release 7のリリースノートです． 本文書が含まれているリリースパッケージについての解説です．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.2） です．

1. 変更点

R7.2での変更点は以下の通りです．

  • 認証方式を強化． 詳細はコラム: 認証方式をごらんください．
  • $SYS_POSTERMAILが1（メイルアドレスを入力しないと書き込みできない）になっている時に，ユーザ設定変更画面で登録メイルアドレスが変更できない，というバグを修正．
  • トップメニューに'HELP'を追加．
  • キーボードショートカットが表示されない，というバグを修正．
  • 訂正済みのメッセージを再度訂正した場合，大元のメッセージが（ディスク上からも）消えてしまう，というバグを修正． 訂正されても消さないつもりでコーディングしてたんですが，このバグのせいで，最大2世代分（現在有効なメッセージとその1世代前のメッセージ）しか保存されていませんでした．
  • メッセージを引用してリプライを書き込む際，先頭行が連結してしまう，というバグを修正．

認証方式の変更がありましたが，ユーザ管理用のkb.userファイルについては上位互換性が保たれています． R7.0〜R7.1.1で使われていたkb.userをR7.2で利用する場合，各ユーザが次回ログインした際に，そのユーザの認証方式が自動的に新方式に移行されます． それまでは旧方式での認証が行なわれます． つまり，R7.0〜R7.1.1でHTTP-Cookiesを利用して認証を行なっていたユーザについては，次回ログインするまで旧方式で認証が行なわれることになります．

R7.1からの変更点の詳細については， Changes: 変更履歴 を参照してください．
