0. このドキュメントの目的

このドキュメントは，KINOBOARDS version 1.0 release 7が動作する環境について説明したものです．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.2） です．

1. 動作要件概略

基本的に，CGIとPerlが利用可能なプロバイダ，もしくはウェブサーバであれば，どこでも動くように作られています． Perlは，perl4でもperl5でも構いません．

2. 動作要件詳細

より詳細には，KINOBOARDS version 1.0シリーズは，以下の環境での動作実績があります． release 7でも，同様の環境であれば動作するものと思います． もし以下の組み合わせで動作しなかったり，これ以外の組み合わせでの動作を確認された方がいらっしゃいましたら，作者まで連絡してください．

  • httpサーバアプリケーション
    • Apache/*
    • IIS/*(with WinNT)
    • Netscape-Commerce/*
    • Netscape-Communications/*
    • Netscape-Enterprise/*
    • Netscape-FastTrack/*
    • Oracle Web Listener/*
    • NCSA/*
    • AN HTTPd(with Win9X/NT)
    • OmniHTTPd/1.0b4(with Win9X)
    • Personal Web Server(with Win9X)
    • QuidProQuo/1.0(with MacOS)
    • MacHTTP/2.2(with MacOS)
    • Cobalt Cube 2700J
    • Cobalt Raq J R15 125 NIU
  • httpクライアントアプリケーション
    • Mozilla/*
    • Internet Explorer/*
    • lynx/*
    • NCSA Mosaic/*
  • perl
    • version 4.036
    • version 5.001
    • version 5.002
    • version 5.003
    • version 5.004
    • version 5.005
    • version 5 for win32
    • MacPerl 5.1.2r2

3. 注意

動作環境について，以下の点については注意してください．

  • Perlのversion 4.019では動きません．
  • emacs/* + w3.el/*では表示が化けます． w3.elが「Content-type」HTTPヘッダの「charset」を，解釈し間違っていることによるもののようです． w3.elを利用する方の多い環境では，cgi.plの中で 
        Content-type: text/html; charset=...
    と出力している部分で，「; charset=...」以降を削除してください．
