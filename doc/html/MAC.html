Mac で、KINOBOARDS を動かそう

KINOBOARDS とは、NAKAMURA Hiroshi さんが、free software として公開されている、Perl でかかれた、CGI Script です。
カスタマイズが簡単に出来る、非常に強力な掲示板システムです。
最小限の変更で、Macintosh 上でも動作する事が分かりましたので、以下にその方法を記します。
（R3.2以降では、マルチ・プラットフォーム対応になりました）

Mac で、CGI を動かす準備

Mac で、CGIを使用可能な WWW Server を用意する。
（Quid Pro Quo 1.0 で動作確認）

MacPerlを、ftp://ftp.lab.kdd.co.jp/lang/perl/CPAN/ports/mac/ などから入手する。
（MacPerl5.1.1r2 で動作確認、現在、5.1.3r2 で稼働中）

MacPerl のヴァージョンが、5.1.1r2 以前の場合、
CGI glue (PCGI)　を
ftp://err.ethz.ch/pub/neeri/MacPerl/
にいってとってくる。
（PCGI_06Feb96 で動作確認）
この中に入っている、　
"MacHTTP CGI Script"　を、
MacPerl の　 MacPerl Extensions　folder　 にほおりこむ。
（MacPerl5.1.2r2 からは、デフォルトで付いてきます）

MacPerl で、メイル送信サービスを使う準備

R3.3以降では、マックでも、メイル送信サービスが使えるようになりました。
ftp://mors.gsfc.nasa.gov/pub/MacPerl/Scripts/ にいって、
libnet を取ってきてください。
（libnet 1.04 で動作確認）

ファイルを解凍後、フォルダ− Net の中身を、
MacPerl フォルダの中の、:lib:Net フォルダ−の中（lin フォルダーの中の、更に、Net フォルダーの中）に、総べて入れてください。

KINOBOARDS のインストール

KINOBOARDS ver.1.0 R3.2 以降を取ってきて、解凍する。
すべてのファイルの改行コードを、Unix の改行コード(LF)から、
マックの改行コード(CR)に変換する。
漢字コードは、EUC のままで、変換しない。

SRAのftp site　から、
jcode.pl-2.0 を取ってきて、改行コードをマック用に変換した後、
jcode.pl という名前で、kb.cgi と同じフォルダ−に入れてください。
jcode.plは(株)SRAの歌代さんが作られたフリーウェアで、
これがなければ日本語のコード変換ができません。

kb.phを書き換えて，きのぼーずをあなたのサイト用に設定し直してください、
KINOBOARDS ver.1.0 R3.2 からは、マルチ・プラットフォーム対応になりました。
kb.phの中の、

  • 管理者の名前　($MAINT_NAME)
  • 管理者のメイルアドレス　($MAINT)
  • 掲示板システムの名前　($SYSTEM_NAME)
  • サーバマシンのタイプ　($ARCH)
  • （メイル送信サービスを使用するならば）メールサーバーのホスト名　($MAIL2)
を、それぞれ、自分の環境に合わせて書き換えてください。管理者の名前とメイルアドレスは，忘れずに変更してください。
サーバーマシンのタイプとして、Mac を選ぶだけで、MacPerl で動くようになります。
これだけでオーケー。
う〜む、そごい。

CGI として機能させる

kb.cgi　 を、MacPerl にドロップし、
Save as で、 "(MacHTTP) CGI Script" を選択する。

メモリが足りないと怒られた時は、MacPerl 本体へのメモリの割り当てを増やしてみてください。

Document をよく読む　(^^;

　これで、魔苦で動かすための準備は出来ました。
　あなたのサイトの /kb/index.html から、KINOBOARDS にアクセスしてみましょう。
　無事動きましたでしょうか？
　あとは、ついてくるドキュメントを参考にしてください。
