1. きのぼずとは?

きのぼず(KINOBOARDS: Kinoboards Is Network Opened BOARD System)は，WWW上で自由に意見交換を行なうことを目指して作成した掲示板システムです． より具体的には，以下を支援することを目的としています．

  • WWWの仕組みを用いて自由に意見交換を行なう．
  • そこで得られた知見をまとめ挙げ，後に役立てる．

つまり，一般的に「掲示板」と呼ばれている，出会いとコミュニケーションが第一目的の，メッセージ書き捨て型のシステムとは，目指しているところが異なります． そのため，きのぼずをチャットシステムの延長として捉えると，やや扱い辛いかもしれません． 逆に，メッセージが書き捨てられていくのに疲れてしまった方にとっては，良い薬になる可能性があります．(^-^

フリー・ソフトウェア（「フリーウェア」とは違い，無償なだけでなく，「自由に」使えるソフトウェア）です． ダウンロードして「自由に」使ってやってください．

2. 機能概要

きのぼずが管理する掲示板の概略は，以下の通りです．

「きのぼず」
「きのぼず」は，1つのシステムで複数の「掲示板」を運営することができます．
「掲示板」
「掲示板」は，その掲示板の「ユーザ」が意見交換を行なうための場を提供します． 「掲示板」には，複数の「スレッド」が含まれます．
「スレッド」
「スレッド」は，その掲示板の「ユーザ」が，とある話題について意見交換するためのまとまりです． 「スレッド」は，複数の「メッセージ」から成ります．
「メッセージ」
「メッセージ」には，以下の2種類のものがあります．
  • 「スレッド」の最初に置かれる，問題提起，話題提供，ネタ振り，提案などをする「オリジナル」
  • 他の「メッセージ」への返事として書き込まれる「リプライ」 （他のシステムでは，レス，フォローなどと呼ばれることもあります）
「ユーザ」
「ユーザ」は，その掲示板への参加者です． 以下で説明するように設定次第ですが，インターネット上の任意のユーザが参加できるようにも，イントラネット内の登録ユーザのみ参加できるようにもできます．

次に，きのぼずが提供する多彩な機能について説明します． 以下の多くの機能について，システム管理者の設定次第で， その機能を利用するかどうか指定できます． 書いていたら結構長くなってしまったので，ここを細かく見るより，実際に動かしながら遊んでみたほうがわかり易いかもしれません．^^;

メッセージ表示機能
きのぼずには各種表示機能が揃っており，ユーザに好みの表示形式を選んでもらえます． ただし，多過ぎてユーザが迷ってしまうという意味では，弱点でもあります．^^;
  • メッセージのタイトルを，まとめて一覧することができます． ユーザは以下の表示方式を自由に組み合わせて表示できます． システム管理者は表示方式のデフォルトを設定できます．
    • スレッド順，もしくは日付順に並べ替えられます．
    • 1ページに表示する件数を指定できます．表示しきれない場合はページ分割され，ページをめくるように表示を切り替えられます．
    • 昇順，降順を入れ替えられます．
    • (*)リプライメッセージを隠して，各スレッドをコンパクトに表示できます．
  • タイトルのみでなく，本文も一緒に表示し，まとめて一覧することができます． タイトル一覧の表示同様，各種表示方式を自由に指定できます．
  • スレッドのメッセージのみを個別にまとめて一覧することができます．
メッセージ登録機能
きのぼずは安全なメッセージ登録機能を提供します．
  • 新規にメッセージを書き込めます． また，各メッセージに対する返事として，リプライメッセージを書き込むことができます．
  • リプライメッセージを書き込む際，リプライ元のメッセージが自動的に引用されます．
  • メッセージに自動的にハイパーリンクを埋め込むことができます．
  • 登録するメッセージにアイコンを貼り付けることができます． 登録内容やリプライ元メッセージとの関係などに応じて貼り付けたり，キーワード替わりに用いるなどすれば，メッセージ群の分類整理に効果的です．
  • 利用可能なアイコンは，掲示板毎に設定することができます．
  • 誰かがリプライメッセージを書き込んだ時，お知らせメイルを送るように指定できます．
  • 書き込んだ後の表示をチェックするためのプレビュー画面があります． XHTMLタグなどの書き込みミスを減らすことができます．
  • メッセージの書き込みフォーマットを自由に選ぶことができます． 書き込み形式は，「そのまま入力(要改行)」，「XHTMLに自動変換」，「XHTMLで入力」の3種類です．
  • (*)書き込むメッセージは，自動的にXHTMLに変換されます．
  • 書き込むメッセージに含まれるタグは詳細にチェックされ，安全であると指定されたタグのみ残されます． このような「安全なもののみ許可」方式は，一般に行なわれている「危険なもののみ不許可」方式に比べ，はるかに安全です．
  • (*)自分が書き込んだメッセージを削除することができます（システム管理者に許可されている場合に限ります）．
  • (*)自分が書き込んだメッセージを訂正することができます（システム管理者に許可されている場合に限ります）．
  • (*)システム管理者のみが新規にメッセージを書き込めるように設定できます（一般ユーザはリプライのみ書き込むことになります）．
メッセージ検索機能
きのぼずは多様な検索方式を提供します．
  • キーワードを指定して，メッセージのタイトル，名前，本文を検索することができます．
  • 複数のキーワードを指定したAND検索が可能です．
  • 書き込み日時の範囲を指定して検索することができます．
  • (*)メッセージが持つアイコンを頼りに検索することもできます． 指定したアイコンを持つメッセージのみでなく，そのアイコンをスレッド中に含むもの，そのアイコンがスレッドの先頭にあるもの，といった検索もできます．
  • (*)検索した結果は，メッセージ単位で表示するだけでなく，スレッド単位でまとめて表示することもできます．
(*)パーソナライズ機能
きのぼずは各ユーザの情報（名前，パスワード，メイルアドレス，Home pageのURL）を記憶し，メッセージ登録時の入力の手間を省きます． HTTP-Cookiesを利用できない環境でも，同一の機能を提供します（システム管理者もしくは各ユーザが，環境に合わせて選ぶことができます）．
  • ユーザ情報の新規登録および登録内容の変更が可能です．
  • 登録済みユーザについては，登録したユーザ情報を利用した認証を行ないます． 未登録のゲストユーザでの参加も可能です．
管理機能
システム管理者向けの機能が豊富なのも，きのぼずの特長です．
  • (*)掲示板を新設することができます． また，使われなくなった掲示板の利用を休止することができます．
  • メッセージを削除することができます．
  • (*)メッセージを訂正することができます． 書き込み日時も訂正できるので，とりあえずメッセージを登録しておいて，後から資料として整理するのに便利です．
  • メッセージのリプライ先を変更できます（長くなったスレッドの分割・再結合も簡単です）．
  • メッセージの表示順序を変更できます（重要なメッセージを目立つ場所へ）．
  • (*)メイリングリスト参加者を追加・削除・編集できます．
  • (*)システム管理者のパスワードも，ブラウザから設定できます（初回起動時に設定します）．

上記において(*)で示した部分が，KINOBOARDS version 1.0 release 7において，前バージョン（release 6）から追加もしくは変更された機能です．
