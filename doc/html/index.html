KINOBOARDS version 1.0 release 2.3

Last Update: 8/22/96

1. 機能概要

きのぼーず(KINOBOARDS: Kinoboards Is Network Opened BOARD System)は， WWWの仕組みを用いて自由に意見交換を行なうことを目指して作成したシステムです． 基本的な機能として，

  • 新規メッセージの書き込み
  • リプライメッセージの書き込み
  • メッセージの参照
  • メッセージ一覧の表示(リプライ順ソート，日付順ソート)
を実装しています． 現在のバージョン ではこれに加えて，
  • 最新のメッセージのまとめ読み
  • あるメッセージへのリプライ記事のまとめ読み
  • リプライメッセージ書き込み時の，元メッセージの引用
  • メッセージの内容に応じたアイコンの表示
  • メッセージの検索
  • 頻繁に書き込む人のためのエイリアス機能
  • 書き込んだメッセージに応答があった時のメールでの連絡
の各機能を持たせました．
