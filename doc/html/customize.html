0. このドキュメントの目的

このドキュメントは，きのぼずのカスタマイズ方法について説明するためのものです．

本文書が対象としているKINOBOARDSのバージョンは， KINOBOARDS version 1.0 release 7（KINOBOARDS/1.0R7.3以降） です．

以下のカスタマイズ方法毎に説明します．

1. 機能
機能が多ければよい，というものではありません． 必要な機能のみ使えるようにカスタマイズしましょう．
2. 見た目
見た目をカスタマイズします．

掲示板毎に変えたい場合は．．．

UI定義ファイル

タグの配置を変えたい場合や画像を張り込みたい場合など，スタイルシートによる変更だけでは物足りないこともあるでしょう．

きのぼずが表示する画面の見た目は，UIディレクトリの中に収められているXMLファイル（以後UI定義ファイルと呼ぶことにします）を変更することにより，カスタマイズすることができます． UI定義ファイルは基本的にXHTMLそのままなので，適宜書き換えてください．

UI定義ファイルには，きのぼず固有のタグがいくつか入れられています． きのぼずはこのタグを見て，ジャンプメニューやメッセージのスレッドなどを合成し，最終的な画面を生成しています． XHTMLのタグを書き換える以外に，きのぼずタグを書き換えるのも，見た目の変更に有効かもしれません．
