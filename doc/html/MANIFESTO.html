Manifesto

Last Update: April 5, 2000

きのぼずパッケージに含まれているファイルの簡単な解説です．

README
ろくな情報ではありませんが，一応，最初に読んでください．
COPYING
GNU GENERAL PUBLIC LICENSEです（英語）． 本パッケージの利用規定になります． 日本語訳がredistディレクトリに含まれていますので，参考にしてください．
doc/
ドキュメントを収めたディレクトリです． 今のところ（将来も? ^^;）HTML形式のドキュメントしかありません．
html/
HTML形式のドキュメントが含まれています．
kb/
CGIプログラム用パッケージを収めたディレクトリです． このディレクトリをウェブサーバに置いて（転送して）ください． その他のファイルはウェブサーバに置かれていなくてもかまいません （稼働時には使われません）．
kb.cgi
きのぼずのプログラム本体です． CGIとして実行するのは，このファイルだけです．
kb.ph
きのぼずの各種設定ファイルです． 表示項目や利用可能機能など，詳細な設定を行なえます． 本ファイルを書き換えると，そのままきのぼず本体の挙動が変更されます．
cgi.pl
CGIプログラム用の各種ユーティリティライブラリです．
jcode.pl
歌代さんによる，日本語文字encoding scheme変換ライブラリです．
mimer.pl, mimew.pl
生田さんによる，MIME base64 encoding/decodingライブラリです．
kinologue.pl
ログファイル作成用のライブラリです．
log/
ログファイルが作成されるディレクトリです．
UI/
きのぼずの各画面表示用プログラムを収めたディレクトリです． プログラム本体はkb.cgiの1つだけなんですが，きのぼずでは， kb.cgiからこのディレクトリ内のファイルを読み込むことによって， 各表示画面を生成しています．
kinoboards
利用掲示板を設定するためのファイルです． 配布時点では，テスト用掲示板のみが登録されています． 新規掲示板の作成方法については， MAINTENANCEドキュメント を御覧ください．
kinousers
ユーザ情報を格納するためのファイルです． 現在ではほとんど使われないんですが，一応残しておいてください．
board/
きのぼずの掲示板ディレクトリの，雛型です． 新規掲示板を作成する場合は，このディレクトリはそのまま残しておき， 新規ディレクトリにコピーしてそちらを書き換えてください．
kb.aid
その掲示板に関する個別情報を格納するファイルの，雛型です．
kb.board
掲示板のヘッダ部分に表示するメッセージを格納するファイルの，雛型です．
kb.db
掲示板に投稿されたメッセージのサマリを格納するファイルの，雛型です．
test/
配布時点でセットアップ済みであるテスト用掲示板のための， 掲示板ディレクトリです． 上記雛型であるboardディレクトリと同じものが入っています．
index.html, index.shtml
掲示板一覧を表示するためのHTMLファイルです． きのぼずには掲示板一覧を表示する機能もあるんですが， デザインに凝りたい方は，こちらのHTMLファイルを利用してください． 掲示板一覧の表示方法については，kb.phの中で説明されています．
icons/
きのぼずが利用する各種アイコンを収めたディレクトリです．
tools/
きのぼず本体からは利用されない（なくても特には問題ない）， しかしきのぼずに関連するツール群を収めたディレクトリです．
conv.cgi
きのぼずのR5.6未満のバージョンからR5.6以降のバージョンに移行する際に， 各種ファイルの更新（具体的にはファイル名変更）を行なうためのツールです． 利用方法は， INSTALLドキュメント を参照してください．
kbTst.cgi
綾乃介さん(aya@big.or.jp)が作ってくださった， きのぼずの実行環境をテストするためのCGIプログラムです． 利用方法は， INSTALLドキュメント を参照してください．
release.rb, getRevisions.rb
なひのリリース作業用のRubyスクリプトです．
redist/
他パッケージから再配布しているファイルを収めたディレクトリです． このディレクトリ以下のファイルの再配布については， 各パッケージで規定されている条件に従ってください．
gpl_text.txt
GNU GENERAL PUBLIC LICENSEの日本語訳です．
jcode.pl
歌代さんによる，日本語文字encoding scheme変換ライブラリです．
mime_pls-2_00alpha_tar.gz, mimer.pl, mimew.pl
生田さんによる，MIME base64 encoding/decodingライブラリです． mime_pls-2_00alpha_tar.gzがフルパッケージになります．
application.rb, deffile.rb
なひ作のRubyライブラリです．きのぼず本体では使いません．
