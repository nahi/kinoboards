Last Update: Feb 18, 1999

※基本的に， 全てのファイルのJapanese Kanji Character Setのencoding scheme (以下単に漢字コード)は日本語EUCです． プログラムの挙動がおかしくなったら，まずは漢字コードをチェックしてくださいね． ファイル名の後ろに書かれているのは， サーバがUNIXの場合のファイルの推奨パーミッションです．

kb/: 777
以下のファイルが含まれているKINOBOARDS/1.0のメインディレクトリ．
kb/COPYING: 444
GNU GENERAL PUBLIC LICENSE Version 2, June 1991のCopyingメッセージ．
kb/README: 444
READMEファイル．最初に読んでください．
kb/UI/: 755
きのぼーずシステム本体のうち， ユーザインタフェイス部のインプリメンテーションを含むディレクトリ． *.plというファイル個々で，各表示画面と送信メイルを生成しています． 画面のタグを変更したい，あるいは送信するメイルの文面を変更したい場合には， 各インプリメンテーションを書き換えてください． ただし，漢字コードは必ずEUCにしてくださいね．
kb/board/: 777
新規に掲示板を作成する際に利用する各種ファイルを保管するディレクトリ． このディレクトリ内は書き換えず， 各ファイルを新しいディレクトリにコピーして，新しい掲示板を作成してください．
kb/board/kb.db: 666
掲示板に書き込まれた記事情報を管理するDB．
kb/board/kb.aid: 666
掲示板に書き込まれた最新記事のIDを管理するDB．
kb/board/kb.board: 644
掲示板の記事一覧の先頭に挿入されるhtmlファイル．
kb/cgi.pl: 444
CGI関連package． kb.cgiで読み込んで， 表示，セキュリティチェック，メイル送信，システムのロック等を行います．
kb/doc/: 755
htmlによるドキュメントが格納されたディレクトリ． doc/index.htmlは， 少なくとも一度はアクセスしてみてください．
kb/icons/: 755
アイコン用gifファイルを保管するディレクトリ．
kb/icons/all.idef: 644
アイコンメッセージとgifファイルとのdefaultの対応を保持します．
kb/icons/{掲示板ディレクトリ名}.idef: 644
各掲示板固有の，アイコンメッセージとgifファイルとの対応を用意したい場合は， このようなファイルを作成してください． 例えばテスト用掲示板では，icons/test.idefが存在するなら， そこに書かれた対応に従うことになります．
kb/index.html: 644
各掲示板に分岐するためのhtmlページ． 自由に書き換えて結構です．このファイルのみ，漢字コードは任意ですが， 読者の利便を考えるとJIS(ISO-2022-jp)にしておいた方がいいと思います (掲示板の中では全てJISになります)．
kb/index.html.cgi-bin: 644
CGI scriptとシステムを別ディレクトリに置かなければならないサイトのための， サンプルindex.htmlファイル．詳しくは INSTALLATIONファイル を参照してください．
kb/jcode.pl: 444
歌代さんによる，日本語漢字コード変換用のpackageです．
kb/kb.cgi: 755
きのぼーずシステム本体であるperl script． アプリケーションモデルとデータモデルがインプリメントされています． ユーザインタフェイス部分はUIディレクトリの下の 各ファイルでインプリメントされます．
kb/kb.ph: 644
きのぼーずシステムのサイト依存の設定を行なうためのperl header． 管理者のメールアドレス，システムの機能設定，メッセージの定義等を行ないます．
kb/kinoboards: 644
きのぼーずシステムが扱う， 掲示板のディレクトリ名と正式名との対応を管理するDB． 掲示板を増やす場合に， ディレクトリ名と掲示板名をTAB文字で区切り，1行に1つの掲示板情報を書き加えます． 行末には必ず改行コードを入れてください． 配布時に登録されている掲示板は，「test/テスト」のみです．
kb/kinologue.pl: 444
ログを書き出すためのpackage． kb.cgiで読み込んで使われます．
kb/kinousers: 666
ユーザのエイリアスを管理するDB． エイリアス名，ユーザ名，E-mail addr.，URL等を管理します． kinoboardsとは異なり， 基本的にユーザが直接ファイルを書き換える必要はありません． 配布時は空です．
kb/mimew.pl: 444
kb/mimew.pl: 444
生田さんによる，MIME base64 encoding/decodingのためのpackageです． mime_pls/2.00αのパッケージに含まれているライブラリです． mime_pls-2_00alpha_tar.gzが，mime_plsのフルパッケージになります．
kb/test/: 777
配布時に用意されているテスト用掲示板の記事を保管するディレクトリ．
